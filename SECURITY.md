# Security

If you believe you have found a security vulnerability in this project, please **do not** open a public GitHub issue.

Instead, report it responsibly by contacting the maintainers. **Maintainers:** add a dedicated security email or GitHub *Security > Private vulnerability reporting* for this repository before announcing it widely. Include steps to reproduce and affected versions if known.

**Out of scope for this public repository:** production API keys, proprietary deployments, embedding/RAG pipelines, and closed-source training or batch jobs are maintained separately; do not include secrets in reports or in sample code.
